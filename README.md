# Jithmi Shihara | Personal Portfolio

A high-end, interactive personal portfolio website built with **React**, **TypeScript**, and **Tailwind CSS v4**. This project showcases my dual expertise as a Frontend Engineer and UI/UX Designer, featuring a modern "Bento Box" architecture and a professional Onyx & Indigo aesthetic.

![Portfolio Preview](https://images.unsplash.com/photo-1517694712202-14dd9538aa97?q=80&w=1200&auto=format&fit=crop)

## ✨ Key Features

- **🚀 Modern Tech Stack:** Built with React 19, TypeScript, and the latest Tailwind CSS v4.
- **🍱 Bento Grid Layout:** Sophisticated grid system for projects and skills, optimized for visual hierarchy.
- **🌗 Advanced Dark Mode:** Custom-built toggle with smooth transitions and high-contrast accessibility.
- **✨ Micro-Interactions:** Fluid animations powered by `framer-motion` and an interactive designer cursor.
- **📱 Fully Responsive:** "Mobile-first" design ensuring a premium experience on every screen size.
- **🎨 UI/UX Focused:** Grainy textures, glassmorphism navigation, and adaptive color systems.

## 🛠️ Built With

- **Frontend:** [React.js](https://reactjs.org/) (v19)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) (v4)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Tooling:** [Vite](https://vitejs.dev/) & [TypeScript](https://www.typescriptlang.org/)

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/JithmiShihara/JimWeb.git
   ```
2. Navigate to the project folder:
   ```bash
   cd JimWeb
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## 📝 Personalization

To update the content:
1. Open `src/data.ts` to modify projects, skills, or personal info.
2. Place your professional photo in `public/images/jithmi.jpg`.
3. Place your resume in `public/resume.pdf`.
